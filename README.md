######동국대학교 공개SW 프로젝트
=============================

####6조 가짜 프로그래머
---------------------------

WebRTC Project 

## 사용법

Node.js 로 Singaling Server 사용
실행하고 하려면

npm install    
npm start


웹 페이지에서 
http://localhost:8443/
입력 후 실행


* * *
현재 Streaming server 미적용을 local 환경에서만 실행 가능
